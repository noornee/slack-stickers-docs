# How to Install  

You can add SlackStickers Bot to your workspace in a few steps:  

### 🛠️ **Installation Steps**  
1️⃣ Click **[Add to Slack](https://slack.com/oauth/authorize?client_id=YOUR_CLIENT_ID)**  
2️⃣ Authorize the bot  
3️⃣ Start using `/gif` or `@sticker`  

🎉 **You're all set!**
