# How to Install

You can add the bot to your workspace in a few steps:

### 🛠️ **Installation Steps**
1️⃣ Click **[Add to Slack](https://auth.stickersforslack.com)**
2️⃣ Authorize the bot
3️⃣ Start using `/sticker` or `@Sticker for slack`

🎉 **You're all set!**
